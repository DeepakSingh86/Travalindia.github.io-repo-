Import product data from CSV files or load directly from GitHub.

To load images from GitHub:

The images/ folder in your GitHub repository contains product images and CSV files.

The CSV file should contain image URLs in the following format:

  • Full GitHub URL: https://raw.githubusercontent.com/Urvashimahawar/urvashi.github.io/main/images/earrings1.jpg
  • रिलेटिव पाथ: earrings1.jpg (सिर्फ फ़ाइल नाम)

CSV फॉर्मेट:

आपकी CSV फ़ाइल में निम्न कॉलम होने चाहिए (इसी क्रम में):

productId, productName, quantity, price, imageUrl1, imageUrl2, imageUrl3

उदाहरण:

P001,सुंदर झुमके साइज-1,50,599.99,earrings1.jpg,earrings2.jpg,earrings3.jpg
P002,सुंदर झुमके साइज-2,30,149.50,https://raw.githubusercontent.com/Urvashimahawar/urvashi.github.io/main/images/earrings4.jpg,earrings5.jpg,earrings6.jpg
